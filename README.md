# Read-BMP-image
A CPP class able to load a colored BMP image file.

可以读取彩色 BMP 图像的 CPP 类。

## Functions
* open & save BMP file (only 24/32 bit file supported)
* print header info
* discolor
* invert color
* generalize histogram
* equlize histogram
* save histogram

* 打开和保存彩色 BMP 文件（只支持 24 位或 32 位）
* 显示元数据
* 去色
* 反色
* 生成直方图
* 直方图均衡化
* 保存直方图
